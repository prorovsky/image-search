Image Search API

User Stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Usage:

  • You can search images.

    https://prorovsky-imgur-api.herokuapp.com/api/imagesearch/lolcats

  • You can through images use offset.

    https://prorovsky-imgur-api.herokuapp.com/api/imagesearch/lolcats?offset=10

  • Or check history of search.

    https://prorovsky-imgur-api.herokuapp.com/api/latest/imagesearch
